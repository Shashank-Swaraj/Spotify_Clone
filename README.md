# 🎵 Spotify UI Clone

A pixel-perfect replication of Spotify's core web interface — built entirely 
with vanilla HTML and CSS. No frameworks, no libraries, just clean semantic 
code.

---

## 📌 Features

- 🎨 Pixel-accurate Spotify UI recreation
- 📚 Sidebar with Home, Search, and Your Library sections
- 🃏 Responsive music cards with hover effects
- ▶️ Bottom music player bar with progress and volume sliders
- 📌 Sticky top navigation bar
- 📱 Partially responsive (hides elements on smaller screens)
- 🎵 Custom range input styling for playback and volume bars

---

## 🛠️ Tech Stack

- **Language:** HTML5, CSS3
- **Fonts:** Google Fonts — Montserrat
- **Icons:** Font Awesome 7

## ⚙️ Installation & Setup

**1. Clone the Repository**
```bash
git clone https://github.com/Shashank-Swaraj/Spotify_Clone.git

```

**2. Open in Browser**
```bash
# Just open index.html directly in any browser
# No build tools or dependencies needed
```

## 💡 CSS Concepts Used

| Concept | Where Used |
|---|---|
| Flexbox | Layout of sidebar, player, cards |
| `position: sticky` | Top navigation bar |
| `position: fixed` | Bottom music player |
| Custom range inputs | Progress bar and volume slider |
| `overflow: auto` | Scrollable main content |
| CSS pseudo-elements | Slider thumb styling |
| `@media` queries | Responsive hiding of elements |
